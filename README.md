
## Advanced Rainfall & Weather Prediction System

A research‑scale weather analytics project using Python and MATLAB.

## Components

- Data ingestion and preprocessing
- Feature engineering
- Machine learning rainfall prediction
- Deep learning rainfall forecasting (LSTM prototype)
- Data visualization
- MATLAB statistical rainfall modeling
- Modular scientific project structure

## Running

Install requirements:

pip install pandas numpy matplotlib scikit-learn

Run pipeline:

python src/main_pipeline.py

## Project Highlights

- Time series rainfall modelling
- Weather variable correlation analysis
- Prediction models
- Visualization modules
- MATLAB climate statistics
